# AI 도구 선택 기준표

## 평가 기준

| 기준 | 세부 내용 | 평가 방법 |
|---|---|---|
| 1. 성능/정확성 | 응답 품질, 환각(오류) 빈도, 작업 적합도 | 동일 프롬프트로 결과물 비교, 점수화(1~5점) |
| 2. 비용 효율성 | 구독료, API 사용료, 무료 티어 한도 | 월 예상 사용량 기준 총비용 산출 |
| 3. 사용 편의성 | UI/UX, 학습 곡선, 통합 기능(연동성) | 직접 사용 후 체감 난이도 평가 |
| 4. 보안/데이터 정책 | 데이터 저장 여부, 기업용 보안 정책, 저작권 이슈 | 공식 정책 문서 확인 |

## AI 도구 평가표

> 각 항목을 1~5점으로 채점 (1=매우 낮음, 5=매우 높음)

| AI 도구 | 성능/정확성 | 비용 효율성 | 사용 편의성 | 보안/데이터 정책 | 총점 |
|---|---|---|---|---|---|
| ChatGPT | 4 | 5 | 4 | 5 | 18 |
| Gemini | 4 | 5 | 4 | 5 | 18 |
| Claude | 4 | 5 | 5 | 5 | 19 |

## 총점 계산 방법

총점 = 성능 점수 + 비용 점수 + 편의성 점수 + 보안 점수 (각 항목 1~5점 합산)

## 메모 / 비고

- ChatGPT:정확성(4)은 대체로 안정적이나 가끔 최신 정보나 세부 사실에서 오류가 발생함. 비용(5)은 무료 티어와 유료 플랜의 가성비가 우수함. 편의성(4)은 UI가 직관적이지만 부가 기능 활용도는 보통 수준. 보안(5)은 기업용 정책이 잘 갖춰져 있음.
- Gemini:정확성(4)은 검색 연동 덕분에 최신 정보 반영은 강하나 답변 일관성이 다소 떨어짐. 비용(5)은 구글 생태계와 통합되어 합리적임. 편의성(4)은 구글 서비스 연동은 편리하나 단독 사용 시 학습 곡선이 있음. 보안(5)은 구글의 데이터 정책을 따라 안정적임.
- Claude:정확성(4)은 추론과 긴 문맥 이해에 강점이 있으나 실시간 정보는 제한적임. 비용(5)은 플랜 대비 품질이 우수함. 편의성(5)은 응답의 가독성과 작업 흐름이 매끄러움. 보안(5)은 데이터 보호 원칙이 엄격하게 적용됨.
그래서 가장 점수가 높고 파일을 만들어 달라고 요청했을 때 파일을 따로 다운받을 수 있게 만들어줘 편의성이 높았던 클로드를 선택했다
