# AI 도구 선택 기준표

## 평가 기준

| 기준 | 세부 내용 | 평가 방법 |
|---|---|---|
| 1. 성능/정확성 | 응답 품질, 환각(오류) 빈도, 작업 적합도 | 동일 프롬프트로 결과물 비교, 점수화(1~5점) |
| 2. 비용 효율성 | 구독료, API 사용료, 무료 티어 한도 | 월 예상 사용량 기준 총비용 산출 |
| 3. 사용 편의성 | UI/UX, 학습 곡선, 통합 기능(연동성) | 직접 사용 후 체감 난이도 평가 |
| 4. 보안/데이터 정책 | 데이터 저장 여부, 기업용 보안 정책, 저작권 이슈 | 공식 정책 문서 확인 |

## AI 도구 평가표

> 각 항목을 1~5점으로 채점 (1=매우 낮음, 5=매우 높음)

| AI 도구 | 성능/정확성 | 비용 효율성 | 사용 편의성 | 보안/데이터 정책 | 총점 |
|---|---|---|---|---|---|
| ChatGPT | 4 | 5 | 4 | 5 | 18 |
| Gemini | 4 | 5 | 4 | 5 | 18 |
| Claude | 4 | 5 | 5 | 5 | 19 |

## 총점 계산 방법

총점 = 성능 점수 + 비용 점수 + 편의성 점수 + 보안 점수 (각 항목 1~5점 합산)
